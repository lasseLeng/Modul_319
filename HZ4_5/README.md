Die Haupt unterschiede zwischen einem Editor und einer Entwicklungsumgebung sind:

Ein Editor ist wehsentlich leichtgewichtiger, dass bedeutet, dass er wenig speicher platz. Hingegn eine Entwicklungs umgebung benötigt wehsentlich mehr speicher.

In einer Entwicklungs umgebung hat man viele möglichkeiten für das Debugging und die fehler sucher. Ausserdem bieten sie eine Einheitliche gute Struktur. Editor hingegen ist nur dazu da um den Code zu bearbeiten es gibt keine Hilfsmittel.
